# FinCal Innovation: Goal-Based Investment Calculator
Developed for TECHNEX 26 

## Overview
This interactive financial calculator is designed as an educational tool to simplify mutual fund investment concepts. It allows users to estimate the monthly SIP required to achieve future financial goals while accounting for inflation and market returns.

## Features & Compliance
* **Mandatory Logic**: Uses industry-standard formulas for goal inflation and SIP accumulation.
* **Transparency**: All assumptions (inflation, returns, timeframes) are fully user-editable and clearly disclosed.
* **Brand Alignment**: Designed using HDFC brand colors (#224c87, #da3832, #919090) and approved typography (Montserrat).
* **Accessibility**: Built with semantic HTML and ARIA roles to target WCAG 2.1 AA compliance.
* **Responsive UI**: Fully optimized for Desktop, Tablet, and Mobile devices.

## Tech Stack
* **Frontend**: Next.js 15.5.9 
* **Environment**: Node.js 22.11.0 
* **Styling**: Responsive CSS with HDFC Brand Guidelines
